Dolt Example
===

# Description

[Dolt](https://github.com/dolthub/dolt) - Tool for use sql with familiar Git CLI. Base on Golang

# Install for Mac

```sh
brew install dolt
```

# How to run mysql compatible server

```sh
dolt sql-server
```

# How to execute sql command

```sh
dolt sql -q "SELECT * FROM user"
```
